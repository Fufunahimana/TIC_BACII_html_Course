Les 3 identités remarquables
Une identité remarquable ou égalité remarquable est une expression mathématique constituée de nombres ou de fonctions polynomiales. Les égalités remarquables sont très utiles pour faire un calcul plus rapide.
On peut distinguer 3 identités remarquables:
La première égalité remarquable: (a+b)2 = a2 + 2ab + b2;
La deuxième égalité remarquable: (a-b)2 = a2 2ab + b2;
La troisième égalité remarquable: (a+b) (a-b) = a2b2,

H20.
